# RoadSafetyTrack Frontend

React + Vite + FHEVM frontend application

## Installation

```bash
npm install
```

## Development

### Mock Mode (Local Hardhat Node)
```bash
npm run dev:mock
```

### Relayer Mode (Sepolia Testnet)
```bash
npm run dev
```

## Generate ABI

After deploying contracts, you need to generate ABI files:

```bash
npm run genabi
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
